Requisitos de qualidade de software

Os requisitos de qualidade de software são, basicamente, requisitos não funcionais. Caracterizam-se por serem quase sempre conflitantes e terem benefícios de difícil mensuração. Sabe-se que, analisando-os como um conjunto, eles são essenciais para garantir a qualidade do software. No entanto, é difícil determinar quanto cada um dos requisitos de qualidade contribuiu para este resultado.

Os requisitos de qualidade são compostos de vários atributos, cada um deles com um valor específico, um peso. Isto é determinado através do conhecimento dos especialistas.
