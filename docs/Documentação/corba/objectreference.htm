Interoperable Object Reference - IOR

A IOR é composta por um Type Name, detalhes de protocolo e endereço e Object Keys. A Object Key é única em um sistema, sendo composta por dois componentes: o nome do adaptador de objeto e o nome do objeto. Observe que uma única IOR pode conter vários pares "protocolo e endereço" e Object Keys. Isso é útil, por exemplo, para criar sistemas tolerantes a falhas.
