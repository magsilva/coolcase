Geralmente temos uma diversidade de objetos que, quando acionados, executam uma mesma ação. Um exemplo seria um botão de Salvar e um ítem no menu Arquivo chamado Salvar. E, claro, isto não foi esquecido no Java Swing. Nele temos as Actions (ações). Os actions são composto de ações e propriedades. Por exemplo, um action chamado Save poderia ter a propriedade Enable e o procedimento para salvar. Se um ítem Salvar do menu e um botão Salvar estivessem associados a este action e o mesmo estivesse disabled, ambos poderiam ficar grayed.

Não são todos os objetos que podem registrar actions. É necessário que eles implementem a ?. Exemplos de algumas classes Swing que o implementam são JMenu, JPopMenu, JToolBar. Para adicionar ações, basta usar o método add(). Adicionando uma ação em um JToolbar, por exemplo, retornaria um objeto da classe JButton que poderiamos então customizar. Podemos também adicionar os botões na nossa JToolBar e acrescentar as ações depois.
